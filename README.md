# Heads Up (Unity Lab, C#)
Mechanics-focused Unity lab project inspired by the *Heads Up* party game format. Built to practice input handling, timing, and UI feedback.

## Features
- Randomized prompt/card system
- Timer-based round loop
- Simple UI for prompts and round state
- Controls:
  - Desktop: Keyboard input (as implemented in the lab)
  - Mobile: Device input (as implemented in the lab)

## Notes
Completed during Fall 2025 as part of the class **Mobile Game Development**
